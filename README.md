# Extensão "EvoluX1n"

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

Este repositório contém o código-fonte da extensão "EvoluX1n", uma poderosa ferramenta desenvolvida para aprimorar a experiência na plataforma Evolua. Com recursos exclusivos, a extensão busca facilitar o uso e personalização da plataforma de aprendizagem.

## Recursos Principais

- **Modo Escuro**: A extensão oferece a opção de ativar um modo escuro elegante, proporcionando uma experiência visual agradável e reduzindo o cansaço ocular durante o uso da plataforma.

- **Pular Aulas**: Com a extensão "EvoluX1n", você pode pular aulas na plataforma Evolua sem restrições. Isso permite que você gerencie seu tempo de estudo de acordo com suas necessidades, adaptando-se a uma rotina mais flexível.

## Como utilizar

1. Faça o download ou clone este repositório para o seu computador.

2. Abra seu navegador e acesse as configurações de extensões.

3. Habilite o modo de desenvolvedor (Developer Mode) nas configurações de extensões.

4. Selecione a opção "Carregar extensão não empacotada" (Load unpacked) e escolha a pasta onde você salvou a extensão "EvoluX1n".

5. Agora você poderá desfrutar dos recursos da extensão "EvoluX1n" ao acessar a plataforma Evolua.
